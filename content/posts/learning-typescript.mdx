# TypeScript Fundamentals Every Developer Should Know

TypeScript has become an essential tool in modern web development. Let's explore some fundamental concepts that every developer should understand.

## Type Annotations

The basics of TypeScript start with type annotations:

```typescript
// Basic types
let name: string = 'John'
let age: number = 30
let isActive: boolean = true

// Arrays
let numbers: number[] = [1, 2, 3]
let names: Array<string> = ['John', 'Jane']

// Objects
type User = {
  id: number
  name: string
  email?: string  // Optional property
}

const user: User = {
  id: 1,
  name: 'John'
}
```

## Interfaces vs Types

Understanding the difference between interfaces and types:

```typescript
// Interface
interface Animal {
  name: string
  makeSound(): void
}

// Type
type Pet = {
  name: string
  makeSound(): void
}

// Interface extension
interface Dog extends Animal {
  breed: string
}

// Type intersection
type DogPet = Pet & {
  breed: string
}
```

## Generics

Generics make components reusable with different types:

```typescript
// Generic function
function getFirst<T>(array: T[]): T | undefined {
  return array[0]
}

// Usage
const firstNumber = getFirst([1, 2, 3])  // Type: number
const firstString = getFirst(['a', 'b'])  // Type: string

// Generic interface
interface Repository<T> {
  get(id: number): T
  save(item: T): void
}
```

## Utility Types

TypeScript provides useful utility types:

```typescript
interface User {
  id: number
  name: string
  email: string
  age: number
}

// Partial - All properties become optional
type PartialUser = Partial<User>

// Pick - Select specific properties
type UserBasics = Pick<User, 'id' | 'name'>

// Omit - Remove specific properties
type UserWithoutEmail = Omit<User, 'email'>

// Required - Make all properties required
type RequiredUser = Required<User>
```

## Best Practices

1. **Use Type Inference When Possible**
```typescript
// Let TypeScript infer the type
const numbers = [1, 2, 3]  // Type: number[]
const user = {
  name: 'John',
  age: 30
}  // Type: { name: string, age: number }
```

2. **Strict Mode**
Enable strict mode in your `tsconfig.json`:
```json
{
  "compilerOptions": {
    "strict": true
  }
}
```

## Conclusion

These TypeScript fundamentals will help you write more robust and maintainable code. As you become more comfortable with these concepts, you'll find TypeScript indispensable in your development workflow.
