# My Development Setup 2024

As a web developer, having an efficient development environment is crucial for productivity. Here's my current setup that helps me stay productive and organized.

## Hardware

- **Laptop**: MacBook Pro M1 Pro 16"
- **Monitor**: LG 27" 4K Display
- **Keyboard**: Keychron K2 Mechanical Keyboard
- **Mouse**: Logitech MX Master 3

## Software & Tools

### Code Editor
- VS Code with these essential extensions:
  - GitHub Copilot
  - ESLint
  - Prettier
  - GitLens
  - Material Icon Theme

### Development Tools
- **Terminal**: iTerm2 with Oh My Zsh
- **Git Client**: GitHub Desktop / Git CLI
- **API Testing**: Postman
- **Database**: TablePlus
- **Browser**: Chrome DevTools

### Productivity Apps
- **Notes**: Notion
- **Todo**: Things 3
- **Password Manager**: 1Password
- **Window Management**: Rectangle

## Terminal Setup

My terminal configuration includes:

```bash
# Custom aliases
alias gs="git status"
alias gc="git commit"
alias gp="git push"
```
